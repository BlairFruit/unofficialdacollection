# Version 2.0.0

Due to this being a major update, I recommend moving from whatever version of the program you're on to this one. Older versions will also not be able to update to this one. More in the notes below.

### Changes

* Program backend was completely redone so it meets basic security standards. I sacrificed this originally for experimenting with updating files, but it wasnt worth it.
* This does mean you can no longer update the app directly from it, instead having to download newer versions every time. Due to this, updates will be less frequent (than they already are)
* Instead, it will now tell you whenever there's a new version on my github.
* It will still automatically update the comic, though. That hasn't changed. :]
* This will be the long-term-support version.
* Added a new app theme that you can use.

### Bug Fixes
* Fixed some bugs related to custom themes, such as the wrong size if windowed.
* Fixed update failures in regards to the comic updater, and made sure it's more reliable.
* Green mode fixed (again)
* Fixed a bug where the app would crash if you put in a specific page address
* Fixed videoplayer bug where it could stop functioning at random

#### Developer Note
*Right now, this is being sent as a pre-release version just so I can iron out the bugs and issues that I ended up missing one way or another. In theory though, it should all work just fine.*
